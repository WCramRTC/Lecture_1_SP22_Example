This is an example of a page that is formatted only using paragraph tags

Each new line is due to a different <p> tag

You can place whatever content you want inside the paragraph tags. Markup language treats the content between the tags as a string.

Use <p> as if you were formatting an essay or using new line on your display in the consoles

Regular new lines and most white space has no visual impact on your page, outside of normal spaces between words

This example has also been placed on github. That's why you can view the page live on the web and also see the code in my hosted repository. Please, take a look.

Because I used a Cascading Style Sheet at one point, I wanted you to see what you will be doing for the future.

This little box has been formatted with the style sheet in the style folder. You're free to clone and play with it.
